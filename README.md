# crt.cr

Bindings and crystal class for libncursesw

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  crt:
    github: maiha/crt.cr
```


## Usage


```crystal
require "crt"
```


## Development

- for ubuntu

```shell
apt-get install libncursesw5-dev
```
